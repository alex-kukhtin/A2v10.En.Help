SheetGroupCell Element

All elements of the sheets

Represents a cell for displaying grouping controls in a sheet.

The cell has no content, instead it displays the grouping controls in the sheet. The appearance depends on the theme of the user interface.

The use of such a cell only makes sense for rows in hierarchical SheetTreeSection. sections. In all other cases, the cell will simply be empty.

Inherits: SheetCell : UIContentElement : UIElementBase
