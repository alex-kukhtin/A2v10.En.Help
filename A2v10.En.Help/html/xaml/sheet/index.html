Elements of the Sheets

  • Sheet - sheet
  • SheetCell - sheet cell.
  • SheetCellGroup - group of the sheet cells.
  • SheetColumn - sheet column.
  • SheetColumnGroup - group of the sheet columns.
  • SheetGroupCell - sheet cell for grouping elements.
  • SheetPage - printed page.
  • SheetRow - sheet row.
  • SheetSection - sheet section.
  • SheetTreeSection - table section for hierarchical data.
  • SheetAutoGenerate - description of automatic generation.
