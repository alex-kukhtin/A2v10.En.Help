SheetAutoGenerate Element

All elements of the sheets

An object that describes the rules for generating a table from data.

Converter: conversion from a string of text to an object is supported, where the given string is a PropertyName, and Mode = FromDataModel.

Properties

Name Type Description
Mode SheetAutoGenerateMode Auto-generation mode. Possible values:
  • FromDataModel - by default. From the data model. The PropertyName property should be set.
PropertyName String Name of the model property that should represent the collection to generate the table.
