Свойства

Имя Тип Описание
Value Object Только привязка (Bind). Привязка к значению модели данных.
ValidateValue Object Только привязка (Bind). Привязка к значению для валидации. Указывается только в случае, если валидатор нужно связывать не со значением в элементе, а с каким-то другим свойством модели.
