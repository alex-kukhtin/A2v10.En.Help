ShadowStyle Listing

All listings

Sets the style of the shadow projected by the item.

Used by: Alert, Block, Grid, Panel, StateCard

Possible values

Value Description
None By default. No shadow.
Shadow1-Shadow5 Options 1-5. The larger the number, the denser the shadow.
