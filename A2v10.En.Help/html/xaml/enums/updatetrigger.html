Перечисление UpdateTrigger

Все перечисления

Задает режим обновления значения модели

Значения

Значение Описание
Default По умолчанию. Blur
Blur Значение в модели обновляется только при потере фокуса элементом или при нажатии клавиши Enter.
Input Значение обновляется по мере ввода символов в поле ввода.
