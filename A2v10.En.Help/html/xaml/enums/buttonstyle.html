ButtonStyle Listing

Sets the style of the command button. The exact look and color of the button depends on the user interface theme.

Used by: Button.

Values

Value Description
Default By default. Gray.
Primary Main button. Dark blue.
Danger, Error, Red Error. Red.
Success, Green Success. Green.
Warning, Orange Warning. Orange.
Info, Cyan Information. Light-blue.
Outline Outline only.
