Элемент IFrame

Все контейнеры

Представляет собой фрейм для загрузки и отображения внешнего независимого документа. Отображается в html тег <irame>.

Наследует: UIElementBase

Свойства

Имя Тип Описание
Source String Url страницы, которая будет загружена во фрейм
Size Size Размер фрейма. По умолчанию - "100% 100%"
Свойства базового элемента
