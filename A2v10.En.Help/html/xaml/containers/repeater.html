Элемент Repeater

Все контейнеры

Не имеет внешнего представления и обрамляющего тега. Используется для вывода повторяющегося содержимого. Другими словами, в разметку просто включается несколько элементов в соответствии с источником данных.

Наследует: UIElement : UIElementBase

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content UIElementBase Элемент, который будет повторен для каждого значения из ItemsSource. Обратите внимание, что привязки внутри содержимого будут относительными. Корень привязки будет представлять собой один элемент из массива ItemsSource.
ItemsSource Array Всегда привязка - Bind. Источник данных для отображения элементов.
