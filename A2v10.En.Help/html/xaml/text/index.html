Text Elements

Text elements (inlines) are used to display text/images.

Abstract Elements

  • Inline - inline element

Text elements

  • Badge - badge.
  • Break - line break.
  • Html - HTML-text.
  • Hyperlink - link.
  • Line - horizontal line.
  • Paragraph - paragraph of text.
  • Popover - link for the popup element.
  • Span - inline element.
  • SpanIcon - icon in text.
  • SpanSum - inline element for amount.
  • StaticImage - image in text.
  • TagLabel - label - tag.
  • Text - text.
