Simple Elements

Simple markup elements generally support converting from a string. This allows for a significant reduction in the markup code.

Simple elements

  • Length - length.
  • Size - size.
  • Thickness - size of rectangular area.
  • GapSize - gap.
  • PrintPage - description of the print page.
  • Accel - accelerator (hotkey).
  • AccelCommand - accelerator command.

Collections

  • AccelCommandCollection - collection of accelerators.
  • UIElementCollection - collection of elements.
  • ResourceDictionary - resource dictionary.
