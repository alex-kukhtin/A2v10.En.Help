Size Element

All simple elements

Задает размер по горизонтали (Width) и вертикали (Height). Хотя элемент представляет собой объект, чаше всего значение задается с помощью конвертора из текстовой строки.

Properties

Name Type Description
Width Length Width.
Height Length Height.

Converter

Conversion from text string is supported. The width and height values are separated by commas. If only one value is specified, it will be used as both width and height.
