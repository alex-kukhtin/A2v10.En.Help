require - загрузить модуль

Глобальные функции

Описание

Функция require загружает модуль.

Синтаксис

require(path)

Параметры

  • {String} path - путь к файлу модуля. Задается относительно корня приложения.

Возвращает

  • Ссылку на экспорт модуля.

Особенности

Стандартные модули не имеют пути. Вместо этого имя модуля начинается с префикса 'std:'.
